### Uber Eats Menu Interface
📁 [`uber-eats-menu/`](./uber-eats-menu/)

A mobile-first, static digital menu interface inspired by the Uber Eats restaurant menu screen. Built with React, Vite, and Radix UI primitives for accessibility and customization.

**Key Features:**
- Restaurant header with Radix UI Toolbar (back, favorite, search buttons)
- Sticky, horizontally scrollable category navigation
- Scroll spy for active category highlighting
- Menu items with images, badges, and dietary information
- Delivery/Pickup mode toggle
- Mobile-first responsive design
- Accessible component structure using Radix UI primitives

**Tech Stack:** React, Vite, Radix UI, CSS
